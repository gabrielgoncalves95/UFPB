# Projeto 2 APA - Merge Sort e Quick Sort - Outubro de 2017

## Para compilar o projeto basta executar o Makefile através do comando (make) e então usar o comando de execução (./sortProgram)

O arquivo de entrada deve estar no mesmo diretório que o executável. Após executado, o programa irá salvar dois arquivos txt (mergeSort.txt e quickSort.txt) no diretório em que se encontra o executável, esses arquivos irão conter os dados de entrada ordenados pelos algoritmos.
